# TareasMVC 📝

**TareasMVC** es una plataforma interactiva de gestión de tareas diseñada para maximizar la productividad personal y de equipo. Construida sobre **ASP.NET Core 8**, la aplicación combina la robustez de una **Web API RESTful** con una interfaz dinámica y multi-idioma, permitiendo un control granular sobre proyectos mediante subtareas y seguimiento visual de progreso en tiempo real.

---

## Funcionalidades Implementadas ✨

* **Gestión Dinámica de Tareas:** Organización de pendientes con indicadores visuales de progreso mediante barras de porcentaje dinámicas.
* **Sistema de Subtareas:** Desglose de tareas principales en pasos accionables con checkboxes interactivos para un seguimiento detallado.
* **Archivos Adjuntos:** Soporte para subir y gestionar documentos o imágenes vinculados directamente a cada tarea.
* **Soporte Multi-idioma:** Interfaz adaptativa con selector de idioma integrado para una experiencia de usuario global.
* **Seguridad y Autenticación:** Implementación de **ASP.NET Core Identity** para el manejo de usuarios, incluyendo soporte para inicio de sesión externo con **Microsoft**.
* **Experiencia UX Moderna:** Uso intensivo de **JavaScript (Fetch API)** para actualizaciones sin recargar la página y **SweetAlert2** para notificaciones interactivas.

## Tecnologías Utilizadas 🚀

* **Backend:** C# / ASP.NET Core 8.0 (MVC + Web API RESTful)
* **Persistencia:** SQL Server con Entity Framework Core
* **Seguridad:** ASP.NET Core Identity & OAuth (Microsoft Account)
* **Frontend:** JavaScript (Fetch API), Bootstrap 5, Animate.css
* **Librerías UX:** SweetAlert2 para alertas y modales dinámicos.
* **Arquitectura:** Consumo de servicios API internos para una interfaz reactiva.
